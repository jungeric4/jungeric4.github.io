The objective was to build a bridge such that it is optimized to achieve maximum load to mass ratio. One 1/8″×3/4″×36″ balsa wood plank and 1/8″×36″ hardwood dowel was used to build a bridge with dimensions 8cm×40cm×10cm. A truss optimizer program was used to minimize the total volume of material while considering different failure modes. Additionally, SOLIDWORKS was used to model each member and the bridge, to which each member was drawn onto AutoCAD for printout using Epilog Mini 24 Laser Cutter. The bridge was then assembled using the template on SOLIDWORKS achieving a final load to mass ratio of 432.74.
